Fortinet – Expertise e Revenda Autorizada

Firewall líder em segurança unificada

A Fortinet foi posicionada como Líder no Quadrante Mágico de UTM do Gartner. As soluções de UTM são, de acordo com o Gartner “soluções multi-funcionais de segurança para redes geralmente utilizadas por empresas de pequeno ou médio porte”.

Os firewalls de UTM são soluções que se encaixam muito bem para empresas com capacidades reduzidas de segurança, já que elas atuam de forma a simplificar o conjunto de soluções presentes na rede, consequentemente facilitando a gestão e otimizando o trabalho das equipes de TI e Segurança.

Proteção total no tráfego interno e externo

Prevenção de intrusão (IPS) e controle de acesso via UTM / NGFW até a sétima camada de rede.

Suporte 24 horas por dia, 7 dias por semana, atualizações periódicas várias vezes ao dia.

Anti-vírus capaz de escanear e bloquear toda e qualquer ameaça que percorre o tráfego na rede gerenciada.

Bloqueio e controle de utilização pelos usuários

Cada usuário segregado em grupos com regras de acesso distintas, controle em tempo real de todos os acessos realizados até a camada de aplicação (Layer 7 / UTM).

Possibilidade de controle por hora e intensidade de tráfego, aplicação de regras de tráfego (shapping) altamente flexível.

VPN / Gerenciamento de múltiplos links de internet

As soluções Fortinet oferecem suporte nativo a VPN Site-To-Site IPSEC / SSL-VPN, conectando usuários e redes entre si, a fins de acesso direto a recursos internos presentes entre elas.

O gerenciamento multi-link da Fortinet garante a resiliência e estabilidade no tráfego interno e externo.

Características

Considerado como um dos principais responsáveis pelo sucesso da Fortinet, o FortiGate™ oferece uma poderosa plataforma de segurança que mistura alta performance e respostas a multiameaças, e avançadas soluções UTM. Adotando tecnologias inovadoras para rede, segurança e análise de conteúdo, o FortGate™ integra um amplo conjunto de tecnologias como: Firewall, VPN, Antivírus, IPS, Filtro WEB, AntiSpam, Controle de Aplicação, Otimização WAN, Alta Disponibilidade (HA), Dual Link, Inspeção de SSL e DLP.

Outra característica diferenciada do Fortigate é o seu sistema operacional FortiOS. Ele acrescenta uma série de recursos de segurança que vão muito além do firewall de rede que combina o ASIC-accelerated stateful inspection com um arsenal de mecanismos de segurança de aplicações integradas para rapidamente identificar e bloquear as ameaças complexas.

Cada FortiGate inclui a mais ampla gama de funções de segurança e networking no mercado:

  • Firewall, VPN, and Traffic Shaping;
  • Intrusion Prevention System (IPS);
  • Antivirus/Antispyware/Antimalware;
  • Integrated Wireless Controller;
  • Application Control;
  • Data Loss Prevention (DLP);
  • Advanced Threat Protection;
  • Contextual Visibility Management;
  • Feature Select with Present;
  • Vulnerability Management;
  • IPv6 Support;
  • Web Filtering;
  • Antispam;
  • VoIP Support;
  • Layer 2/3 Routing;
  • WAN Optimization & Web Caching.

Soluções Complementares:

Além das soluções de alto desempenho da família FortiGate UTM, a Fortinet também oferece uma grande variedade de tecnologias relacionadas para proteger sua organização e simplificar sua infra-estrutura de TI. Eles incluem soluções de acesso à rede, segurança de aplicativos e Gestão.

  • Network Access: Wireless LAN / Wireless WAN / Switches / Endpoint and Mobility / User Identity / Two-Factor  Authentication Token.
  • Application Security: Application Delivery Controllers / Cloud-based Server Load Balancing / Database Security/ DDoS Protection / Link Load Balancer / Secure Mail Gateway / Web Application Firewall / Web Caching.
  • Management: Centralized Log and Reporting / Centralized Device Management  / Hosted Security Management / Configuration Migration Tool / IP Video and Surveillance Systems / Phone Systems and Phones.
